# 硕士阶段的主要项目如下：
## [基于Issac gym的机械臂抓取强化学习项目复现](项目1-基于Isaac_gym的机械臂抓取强化学习项目复现)
主要内容如下：
1. [基于 Isaac Gym 平台，采用 PPO 算法训练 Franka Emika Panda 机械臂完成立方体抓取与堆叠任务](项目1-基于Isaac_gym的机械臂抓取强化学习项目复现/项目展示.pdf)。 
2. 通过设计多阶段奖励函数，引导策略逐步学习并掌握复杂操作行为，同时利用 GPU 并行环境显著加速训练过程。  
3. 负责仿真环境搭建、资产加载（机械臂臂、桌面、立方体）、PPO超参数设计（horizon length、learning rate、KL target等）、训练监控与结果可视化 [策略演示视频](项目1-基于Isaac_gym的机械臂抓取强化学习项目复现/训练结果演示视频.mp4)。  
## [新型机械超材料阻尼器设计和试验](项目2-新型机械超材料阻尼器设计和试验)
主要内容如下：
1. [提出一种基于机械超材料剪切失稳耗能的新型阻尼器，利用结构剪切失稳实现预定非线性力学行为和多级能量耗散，结合有限元仿真、理论分析与 3D 打印试验，对新型阻尼器的力学性能与耗能机理进行了系统验证](项目2-新型机械超材料阻尼器设计和试验/项目展示-新型机械超材料阻尼器设计和试验.pdf)。
2. 利用Abaqus和Python进行有限元建模和数值模拟。
3. 利用Solidwork进行结构设计优化。
4. 利用3D-打印进行试件制备，并进行力学试验。
## [新型建筑结构设计和试验](项目3-新型建筑结构设计和试验)
